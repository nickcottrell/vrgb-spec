# Assets

This directory contains visual aids, diagrams, and supplementary materials for the VRGB specification.

## Planned Content

- Colorspace geometry diagrams (LAB, HSL, RGB visualizations)
- Schema interpretation flowcharts
- Confidence metric illustrations
- Cross-domain interpretation examples
- System architecture diagrams

## Contributing Diagrams

When adding diagrams to this directory:

1. **Format**: Use SVG for vector graphics (preferred) or PNG for raster images
2. **Naming**: Use descriptive kebab-case names (e.g., `lab-colorspace-3d.svg`)
3. **Documentation**: Update this README with a description of each asset
4. **License**: All assets inherit the Apache 2.0 license unless explicitly noted

## Tools

Suggested tools for creating VRGB-related diagrams:

- **Colorspace visualizations**: Python (matplotlib, colorspacious)
- **System diagrams**: Mermaid, Draw.io, Excalidraw
- **Vector graphics**: Inkscape, Figma, Adobe Illustrator
